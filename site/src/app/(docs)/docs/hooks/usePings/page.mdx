# usePings

All super-power of **react-pings** are come from `usePings` hook.

`usePings` is very important hook for this library all the toasts are come from this hook.

## Syntax

```tsx
usePings();
```

## Parameters

- No parameters

## Usage

```tsx
import { usePings } from "react-pings";

const MyComponent = () => {
  const ping = usePings();
  // now ping variable have all the super-powers of usePings hook
};
```
